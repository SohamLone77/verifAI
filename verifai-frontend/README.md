# VerifAI Frontend

React dashboard for VerifAI analytics, ROI insights, and cost tracking.

## Setup

1. Install dependencies

```bash
cd verifai-frontend
npm install
```

2. Start development server

```bash
npm start
```

3. Build for production

```bash
npm run build
```

## Environment Variables

Create a `.env` file:

```env
REACT_APP_API_URL=http://localhost:8000
REACT_APP_WS_URL=ws://localhost:8000
```
